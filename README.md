# Cenit Home

Engine that provides controllers and models for Cenit Home

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'cenit-home'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cenit-home
